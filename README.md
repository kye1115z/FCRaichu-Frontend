## 🛠 Tech Stack

### Frontend

- **Language & Framework**: `React`, `TypeScript`, `Vite`
- **Styling**: `Tailwind CSS`
- **Data Mocking**: `MSW (Mock Service Worker)`
- **Deployment**: `Vercel`

### Backend

- **Core**: `Java`, `Spring Boot`
- **Persistence**: `JPA`, `JPQL`
- **Security**: `keycloak`
- **Build**: `Gradle`
